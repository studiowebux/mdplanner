# MD Planner

Markdown-based project management with directory storage. One `.md` file per
entity, no database, no external dependencies.

## What is MD Planner?

MD Planner stores tasks, notes, goals, milestones, and 40+ other entity types as
plain markdown files with YAML frontmatter. The project directory is the
database. Edit files directly, use the web UI, or connect via MCP or WebDAV.
