## Removed

- **Brain Manager UI** — removed the built-in Brain Manager feature (backend
  routes, frontend module, CSS, CLI flags `--brains-config` and `--claude-dir`,
  env vars `MDPLANNER_BRAINS_CONFIG` and `MDPLANNER_CLAUDE_DIR`). Brain
  management is now handled externally via cerveau.dev.
