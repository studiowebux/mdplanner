### Added

- GitHub integration: list open issues per repository with state and assignee
  filters (`GET /repo/:owner/:repo/issues`, MCP `github_list_issues`).
- GitHub integration: list open pull requests per repository with state filter
  (`GET /repo/:owner/:repo/pulls`, MCP `github_list_prs`).
- GitHub integration: merge a pull request from MDPlanner
  (`PUT /repo/:owner/:repo/pulls/:n/merge`, MCP `github_merge_pr`). Supports
  squash, merge, and rebase strategies.
- GitHub view: expandable repo rows showing inline issues and PRs. Issues
  assigned to the authenticated user are highlighted. Open PRs have a merge
  button.
- GitHub provider: `listIssues`, `listPRs`, `mergePR` methods added to the
  provider interface.
- Issue and PR types now include `labels`, `assignee`, `createdAt` fields.
  PR type adds `headBranch` and `reviewDecision`.
