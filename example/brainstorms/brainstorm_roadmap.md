# Product Roadmap Q2 2026

## What are our top 3 priorities this quarter?

1. Complete v1→v2 migration for all remaining domains
2. Ship CRM module (companies, contacts, deals)
3. Add invoice PDF export and quote approval workflow

## How do we measure success?

Track migration coverage (domains done / total), user adoption of new billing features, and reduction in v1 API calls. KPI: 100% domain migration by end of Q2.

## What's the biggest risk?

Scope creep from feature requests during migration. Mitigation: strict milestone scoping — owner controls what goes into each sprint.

## Who needs to be involved?

Engineering (Claude) for implementation, product owner (Tommy) for prioritization and review, early users for billing workflow validation.
