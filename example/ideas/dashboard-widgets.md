# Customizable Dashboard Widgets

Let users build their own dashboard by adding, removing, and rearranging
widgets. Each widget pulls data from a specific domain (tasks, goals, KPIs).

## Widget Ideas

- Task burndown chart
- Goal progress ring
- KPI sparkline
- Recent activity feed
- Team availability heatmap
