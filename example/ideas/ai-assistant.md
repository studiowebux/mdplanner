# AI Task Assistant

Use AI to help users write better task descriptions, suggest subtasks, and
estimate effort.

## Potential Features

- Auto-generate subtasks from description
- Smart due date suggestions
- Similar task detection
- Natural language task creation

## Considerations

- API costs (OpenAI/Claude)
- Privacy concerns with task content
- Accuracy of suggestions
