# v2 Platform Launch — Feature Prioritization

Scoped for Q2 launch. Reviewed with engineering and product leads.

## Must Have

- Multi-workspace support
- Role-based access control (admin, member, viewer)
- REST API with OpenAPI spec
- Audit log for all mutations
- Data export (JSON + CSV)
- GDPR-compliant data deletion

## Should Have

- SSO via SAML 2.0
- GitHub and GitLab integration
- Webhook delivery for task events
- Advanced filtering and saved views
- Public API rate limiting and quotas

## Could Have

- Zapier connector
- Slack notifications
- Custom branding for enterprise tier
- AI-assisted task description generation

## Won't Have

- Native iOS / Android apps (web-only for now)
- Video conferencing integration
- Built-in invoicing
- On-premise deployment option
