# Personal Systems

## Morning routine
6:30 — wake, no phone
7:00 — review MDPlanner tasks for the day
7:30 — deep work block (no meetings)
12:00 — lunch + walk

## Weekly review (Sundays)
- Close all open tabs
- Review weekly goals vs actual
- Plan next week in MDPlanner
- Update reading list

## Tools in use
- MDPlanner — tasks, notes, goals
- Helix — all text editing
- Obsidian — long-form writing only
- BuJoTui — daily log

## Rules
- One project at a time
- Inbox zero daily
- No meetings before 10am
