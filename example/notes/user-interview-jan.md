# User Interview — January 22

**Participant:** Beta user (freelance dev, solo)

## Pain points with current tools
- Notion too heavy, slow on large workspaces
- Linear misses personal notes / journaling
- No tool links tasks → invoices → time tracking

## What they liked in demo
- Notes + tasks in same tool
- Fast SSR, no JS bundle lag
- Markdown-first editing

## Feature requests
- Mobile view (high priority for them)
- Recurring tasks
- Simple time tracker with hourly rate
