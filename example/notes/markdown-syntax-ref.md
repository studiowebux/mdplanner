# Markdown Syntax Reference

## Headers
`# H1` `## H2` `### H3`

## Emphasis
`**bold**` `*italic*` `‾‾strikethrough‾‾`

## Lists
- Unordered: `- item` or `* item`
- Ordered: `1. item`
- Task: `- [ ] todo` / `- [x] done`

## Code
Inline: `` `code` ``
Block: ` ```language `

## Links & images
`[text](url)` `![alt](url)`

## Tables
```
| Col | Col |
|-----|-----|
| val | val |
```

## Blockquote
`> quoted text`
