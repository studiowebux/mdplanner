# Vim / Helix Keybindings Reference

## Helix (daily driver)
- `gd` — go to definition
- `gr` — go to references
- `space-f` — file picker
- `space-b` — buffer picker
- `C-w s/v` — horizontal/vertical split

## Useful motions
- `ci"` — change inside quotes
- `da(` — delete around parens
- `<count>j/k` — jump N lines

## Macros
Record with `q<letter>`, replay with `@<letter>`.
Apply to visual selection with `:norm @<letter>`.
