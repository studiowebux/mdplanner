# Client Onboarding Template

## Week 1 — Discovery
- [ ] Kick-off call (90 min)
- [ ] Share access to design files
- [ ] Review existing codebase (if any)
- [ ] Set up project in MDPlanner

## Week 2 — Planning
- [ ] Define milestones and deliverables
- [ ] Agree on communication cadence
- [ ] Set up staging environment
- [ ] Sign contract + invoice deposit

## Ongoing
- Weekly progress update (async, written)
- Bi-weekly check-in call (30 min)
- Monthly invoice on 1st

## Offboarding
- Deploy to production
- Handover documentation
- 30-day support period
