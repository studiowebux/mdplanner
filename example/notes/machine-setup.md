# Machine Setup Checklist

## Fresh macOS setup
- [ ] Install Homebrew
- [ ] `brew install git gh deno`
- [ ] Install Nix (`sh <(curl -L https://nixos.org/nix/install)`)
- [ ] Clone dotfiles repo
- [ ] Install Helix from nixpkgs
- [ ] Configure `.zshrc` with aliases
- [ ] Set up SSH key + add to GitHub
- [ ] Clone all active projects

## VS Code / Cursor (backup)
- Extensions: ESLint, Prettier, GitLens
- Settings sync enabled

## Must-have tools
- Raycast (launcher)
- Magnet (window manager)
- TablePlus (database GUI)
