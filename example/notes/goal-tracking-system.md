# Goal Tracking System

## Structure
- **Goals** — quarterly objectives with KPIs
- **KPIs** — metrics linked to goals (revenue, users, tasks done)
- **Portfolio** — projects that advance goals

## Review cadence
- Daily: check task list against goals
- Weekly: update KPI values
- Monthly: full goal review + adjust

## Current Q1 goals
1. Ship MDPlanner v2.0.0 → KPI: 542 tasks done
2. First paying customer → KPI: $1 MRR
3. 10 beta signups → KPI: waitlist count

## What's working
Linking KPIs to goals makes progress visible immediately.
